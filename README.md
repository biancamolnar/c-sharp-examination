Välkommen till Guess My Number spelet!

Spelets mål är att gissa rätt på ett slumpmässigt nummer mellan 0 och 20. Skriv in din gissning i fältet och tryck på "guess" knappen. Du kommer då få se om det numret vi söker är högre eller mindre så du kan fortsätta gissa, eller om du till och med har lyckats gissa rätt!
Om du vill ge upp så skriver du -1. 

Det finns även en highscore som håller koll på dina gissningar. Vill du spela igen så trycker du på "Play again"!

Lycka till och hoppas du tycker det är skoj att spela Guess my number!
